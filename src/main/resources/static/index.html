Om Turistguiden

Denne turistguide giver et overblik over populære turistattraktioner i Danmark. Guiden er primært rettet mod turister og besøgende, som ønsker inspiration til oplevelser i danske byer – særligt København.

Applikationen stiller data til rådighed via et REST API, som kan anvendes af andre systemer, apps eller vises direkte i browseren i JSON-format.
